Id's en Anchors

In de begintag kan je in HTML een id toevoegen. Het is een soort van class, maar deze mogen per pagina meerdere atributen aan 1 class toevoegen. Id's mogen maar 1 waarde tegelijk hebben. Je kan deze daardoor bijvoorbeeld gebruiken voor een auto-scoll van de inhoud naar het aangeklikte onderdeel. Je kan met het a-element verwijzen naar de Id. Dat doe je met een '#' en vervolgens de id-tekst. Dus

Id's en Hyperlinks voor andere plek op pagina. in Css scroll-behavior:smooth; zodat hij smooth scrollt. Scroll-margin:; voor de ruimte boven het aangeklikte element zodat hij niet automatisch helemaal bovenaan de pagina staat, maar bijv in het midden.
@keyframes voor animaties zonder JavaScript. @keyframe naam van animatie{0% {voor start animatie} 100% {voor eind animatie} }
