Hoofdstuk 1 - Wat is HTML?

HTML staat voor HyperText Markup Language. Dat klinkt ingewikkeld, maar het idee is simpel:
HTML beschrijft wat iets is op een webpagina. Bijvoorbeeld:

  • dat is een titel
  • dat is een paragraaf
  • dat is een afbeelding
  • dat is een menu
HTML zegt niet hoe iets eruitziet. Dat doet CSS. En HTML zegt ook niet wat iets doet. Dat doet JavaScript.

Hoofdstuk 2 - Wat zijn HTML-elementen?

HTML bestaat uit elementen. Een element ziet er meestal zo uit:
<p>Dit is een tekst</p>
Dit bestaat uit:

  • een openingstag ( '<p>')
  • de inhoud ('Dit is een tekst')
  • een sluitingstag ( '</p>' )
Samen noemen we dit een HTML-element.
